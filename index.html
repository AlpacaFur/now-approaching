Bitmap Font

This is using Daktronics Full Width 9x15, the same font used on MBTA Orange Line train boards. Thanks to BenTheMiner for help in sourcing the font!

Thanks

Technology

Built using Three.js, GLSL, TypeScript, and Vite. Rendering experimentation started on ShaderToy.

View Options

p - Toggle underlying pixel bitmap

r - Switch between render styles

m - Toggle fish multiplexing

t - Switch between 12-hour/24-hour time
